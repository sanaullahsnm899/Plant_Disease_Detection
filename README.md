# 🌿 Plant Disease Detection using Deep Learning

A professional Deep Learning project for detecting plant diseases from leaf images using Computer Vision and Convolutional Neural Networks (CNN). This system helps identify plant diseases early, improving crop health, productivity, and decision-making for farmers and researchers.

---

## 📌 Project Overview

Plant diseases significantly reduce agricultural productivity and crop quality. Manual disease detection is often slow, expensive, and requires expert knowledge.

This project uses **Deep Learning + Image Classification** to automatically detect plant diseases from leaf images with high accuracy.

The model is trained on plant leaf datasets and can predict diseases efficiently through a simple Python-based interface and can also be integrated into a web app using Streamlit.

## ⚙️ Installation

### Step 1: Clone the Repository

```bash
git clone https://github.com/sanaullahsnm899/Plant_Disease_Detection.git
cd Plant_Disease_Detection
```

### Step 2: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 3: Run the Application

```bash
python main.py
```

If using Streamlit:

```bash
streamlit run app.py
```
